# 00. Project Overview: Omarchy Android Control Center

## Introduction
This project creates a seamless, native-feeling integration between Android devices and an Arch Linux (Omarchy/Hyprland) desktop. It allows for:
1.  **Seamless Launching:** Opening specific phones with a single keybind (`Super + Shift + I`).
2.  **Context Awareness:** Automatically detecting if a Samsung Z Flip 5 is folded or open and adjusting the display target accordingly.
3.  **Wireless Independence:** Using Tailscale to maintain connection regardless of local network changes.
4.  **Headless Integration:** "Replacing" terminal windows or launching purely in the background for a distraction-free experience.

## Architecture
* **OS:** Arch Linux (Omarchy distribution)
* **Window Manager:** Hyprland
* **Core Tools:** `adb` (Android Debug Bridge), `scrcpy` (Screen Copy), `tailscale` (Mesh VPN).
* **Menu System:** `wofi` (or `rofi`) for the device selection overlay.
* **Scripting:** Bash with hardware state detection (`dumpsys`/`cmd device_state`).
