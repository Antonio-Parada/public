# 03. Hyprland & Omarchy Integration

## 1. Keybindings (`bindings.conf`)
We use the Omarchy-specific `bindd` dispatcher to ensure the shortcut appears in the system help menu.

**File:** `~/.config/hypr/bindings.conf` (or `hyprland.conf`)

```ini
# Syntax: bindd = MOD, KEY, DESCRIPTION, DISPATCHER, COMMAND
bindd = SUPER SHIFT, I, Android Control, exec, ~/.local/bin/android-menu
```

*Note: We use the full path `~/.local/bin/android-menu` to avoid PATH environment issues during launch.*

## 2. Window Rules
**File:** `~/.config/hypr/hyprland.conf`

We need rules for the phone windows AND the interactive "Repair" terminal.

```ini
# 1. Phone Windows (Floating, Centered, Square-ish)
windowrulev2 = float, class:^(scrcpy)$
windowrulev2 = size 800 800, class:^(scrcpy)$
windowrulev2 = center, class:^(scrcpy)$

# 2. Repair Wizard Terminal (Floating Dialog)
# Matches the --class="repair_term" set in android-menu.sh
windowrulev2 = float, class:^(repair_term)$
windowrulev2 = size 600 300, class:^(repair_term)$
windowrulev2 = center, class:^(repair_term)$
```
